​🎮 Jokenpô Cyberpunk - VS Javi
​Este é um projeto de Jokenpô (Pedra, Papel e Tesoura) com uma interface futurista inspirada na estética Cyberpunk. O jogador enfrenta o "Javi", uma inteligência artificial que desafia suas escolhas em um ambiente de luzes neon e efeitos visuais imersivos.

​🚀 Tecnologias Utilizadas
*​HTML5: Estrutura semântica do jogo.
*​CSS3: Estilização avançada com foco em Glassmorphism, animações de Neon Flicker e Responsividade via Media Queries.
*​JavaScript (ES6+): Lógica de jogo, manipulação do DOM e controle dinâmico de animações e placares.

​🎨 Diferenciais do Projeto
*​Efeitos de Neon: Uso de text-shadow e keyframes para criar uma atmosfera de luzes vibrantes.
​*Interface Responsiva: O jogo se adapta perfeitamente a diferentes tamanhos de tela, garantindo uma boa experiência tanto no desktop quanto no mobile.
​*Feedback Visual: Animações exclusivas para situações de Vitória, Empate e Game Over.
